#Xblog

免费、开源的博客平台
-----------------------------------

风格设计基于AmazeUI
运行环境要求 Apache + PHP5.6 + MySql

Xblog的安装流程
-----------------------------------

### 1、克隆库到您的生产环境。
    git clome git://git.coding.net/weilantian/Xblog.git
### 2、为Xblog创建一个数据库名字随意，编码utf-8_bin
    create database xblog
### 3、使用浏览器访问：
    http://{生产环境目录}/{xblog目录}
### 4、完成安装向导
### 5、开始使用
